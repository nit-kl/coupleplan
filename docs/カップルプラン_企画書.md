# 📱 企画書：カップルプラン

> **二人の「なんとなく」を、最高の思い出に変える。**
> カップル専用のポータル型Webアプリ

---

## 1. エグゼクティブサマリー

「カップルプラン」は、**カップルの日常の"小さなモヤモヤ"をゲーム感覚で解決する**ポータル型Webアプリです。

1stリリースでは、SNSでのバズと口コミ拡散を狙うため、以下2つのキラーコンテンツを中核として提供します。

| プロダクト | 解決する悩み | バズ要素 |
|---|---|---|
| 🎲 **ミステリールーレット** | 「今週末何する？」問題 | マッチングアプリ風UIで「二人の合意」を可視化 |
| 🥷 **サイレント・ニンジャ** | 家事の押し付け合い問題 | 秘密×貢献度バトルのゲーミフィケーション |

両アプリとも「**スクショしてSNSに載せたくなる**」結果画面を設計し、自然拡散を狙います。

---

## 2. プロダクト概要

### 2.1 サービス名
**カップルプラン（CouplePlan）**

### 2.2 コンセプト
「恋人と過ごす時間を、もっと楽しく・もっとフェアに」

長く付き合うカップル・同棲カップル・夫婦が直面する「マンネリ」「家事分担」「予定決め」といった**日常的なストレス**を、ゲーム化・アプリ化することで楽しく解決します。

### 2.3 提供形態
- **Webアプリ**（PC / スマホブラウザ対応、レスポンシブ設計）
- 1stリリース時点ではネイティブアプリ未対応、PWA対応で擬似アプリ体験を提供

### 2.4 ターゲットユーザー

| ペルソナ | 年齢層 | 関係性 | 主な悩み |
|---|---|---|---|
| **交際中カップル** | 20代〜30代 | 付き合って半年〜3年 | デートがマンネリ、予定決めが面倒 |
| **同棲カップル** | 20代後半〜30代 | 同棲1年以上 | 家事分担の不公平感 |
| **新婚夫婦** | 20代後半〜30代 | 結婚1〜3年 | パートナーへの感謝を忘れがち |

## 3. 1stリリースの戦略

### 3.1 なぜこの2つのアプリか

カップル向けアプリが乱立する中、バズを起こすには「**共感×驚き×共有したくなる仕掛け**」が必要です。

- **ミステリールーレット** → 「わかる！」の共感軸（今週末何する問題は全カップル共通）
- **サイレント・ニンジャ** → 「こんなアプリ待ってた！」の驚き軸（既存サービスに類似なし）

2つを同時リリースすることで、**「デート前〜デート中〜日常生活」**というカップルの時間軸すべてをカバーします。

### 3.2 バズ設計

- **SNS映えする結果画面**：ルーレットの当選結果、ニンジャポイントのランキングをスクショしてX（旧Twitter）やInstagramにシェアしたくなるビジュアル
- **ハッシュタグキャンペーン**：`#ミステリールーレット結果` `#ニンジャポイント報告`
- **インフルエンサー施策**：カップル系YouTuber・TikTokerへの先行提供

## 4. プロダクト詳細

## 4.1 🎲 ミステリールーレット

### コンセプト
> **運命のデートを決める、二人だけのマッチングゲーム。**

「今週末何する？」というカップル特有の悩みを解決するアプリ。マッチングアプリのようにデートのアイデアをスワイプし、お互いが「いいね」したプランの中からアプリがランダムにルーレットで予定を決定します。

### 体験フロー

1. **プラン選択フェーズ**
   - アプリが「水族館」「新しいカフェ巡り」「映画鑑賞」「公園でピクニック」などのデートプランをカード形式で提示
   - カップルそれぞれが**スワイプ**で「いいね／パス」を判定（Tinder風UI）
   - 相手の判定は**見えない**（ブラインド投票で気を遣わない）

2. **マッチングフェーズ**
   - お互いが「いいね」したプランのみが候補として残る
   - 候補が3つ以上揃ったら次へ進める

3. **ルーレット演出フェーズ**
   - 候補プランが入ったルーレットが回転
   - ドキドキ感を演出するアニメーションで予定を抽選
   - 結果発表：「今週末のデートは〇〇に決定！」

4. **シェア＆保存フェーズ**
   - 結果画面をスクショしてSNSシェア
   - カレンダー連携でそのまま予定登録

### 主要機能

| 機能 | 説明 |
|---|---|
| プランカードDB | 編集部選定のデートアイデアを500件以上搭載 |
| カスタムプラン追加 | ユーザー自身がプラン候補を作成可能 |
| 地域フィルター | 現在地や指定エリアに合わせてプランを絞り込み |
| 予算フィルター | 「〜3,000円」「〜10,000円」など予算で絞り込み |
| 履歴機能 | 過去に決定したデートを振り返り |

### KPI
- 週1回以上の利用率（WAU/MAU）
- 1マッチングあたりの所要時間（目標：5分以内）
- シェア率（結果画面のスクショ数）

---

## 4.2 🥷 サイレント・ニンジャ

### コンセプト
> **気づかれたら負け。愛は、こっそり届ける。**

相手に気づかれないように家事（靴を揃える、水回りを拭くなど）や、ちょっとしたサプライズを行い、アプリ内で「ニンジャポイント」を自己申告するゲーム。週末にポイント結果が発表され、勝者は敗者から「肩もみ」などのご褒美をもらえる仕組みで、家事の押し付け合いをポジティブに解決します。

### 体験フロー

1. **任務選択フェーズ**
   - アプリ内の「任務リスト」から実行した家事・サプライズを選択
   - 例：「靴を揃えた（5pt）」「水回りを拭いた（10pt）」「冷蔵庫の中身を整理した（15pt）」

2. **秘密の申告フェーズ**
   - 実行したらアプリから自己申告
   - **相手にはリアルタイムでは通知されない**（サイレントが鉄則）
   - 任意で写真を添付（証拠写真）

3. **週末の結果発表フェーズ**
   - 毎週日曜20時、両者のニンジャポイントを集計
   - 勝敗と「総貢献ログ」が一気に公開される
   - 「え、そんなこともしてくれてたの！？」という**感謝のサプライズ**が発生

4. **ご褒美フェーズ**
   - 敗者は勝者に「ご褒美」を提供（肩もみ・好きな朝食を作る・1時間自由時間をあげる等）
   - ご褒美メニューは事前に二人で設定

### 主要機能

| 機能 | 説明 |
|---|---|
| 任務マスターリスト | 100種類以上の家事・サプライズをDB化 |
| ポイント設計 | 難易度・所要時間に応じてポイント配分 |
| カスタム任務登録 | 二人の生活に合わせた任務を自由に追加 |
| 写真証拠機能 | ビフォーアフター写真で「証拠」を残す |
| ご褒美メニュー | カップルごとにカスタマイズ可能 |
| 週間レポート | 週末にアニメーション付きで結果発表 |
| 連続記録 | 「連続〇週達成」のバッジ機能 |

### 心理的ポイント

- **「気づかれないように」がゲーム要素になる** → やっている側も楽しい
- **結果発表のサプライズ効果** → 「ありがとう」が自然に生まれる
- **ポジティブな競争** → 家事の押し付け合いではなく、貢献の奪い合いに転換

### KPI
- 週間アクティブ率（週末の結果発表参加率）
- 平均ニンジャポイント申告数／週
- ご褒美達成率

## 5. バズ戦略・マーケティング

### 5.1 SNS拡散の仕掛け

| 施策 | 内容 |
|---|---|
| **シェアしたくなる結果画面** | ルーレット結果・ニンジャポイント結果をカード画像化、ワンタップでXに投稿可能 |
| **公式ハッシュタグ** | `#カップルプラン` `#ミステリールーレット` `#サイレントニンジャ` |
| **UGCコンテスト** | 「今週のニンジャポイント自慢」をSNS投稿 → 毎週公式アカウントで紹介 |
| **インフルエンサー施策** | カップル系YouTuber・TikTokerに先行提供、リアクション動画を狙う |

### 5.2 PR訴求ポイント

- **「家事の押し付け合いをなくすアプリ」**（ジェンダーギャップ文脈）
- **「付き合いたてのドキドキを取り戻す」**（マンネリ解消文脈）
- **「言えない"ありがとう"を見える化」**（感謝・コミュニケーション文脈）

### 5.3 ローンチ時期の提案

- バレンタインデー／ホワイトデー前後（2月〜3月）
- 年末年始のカップル需要期（12月）
- いずれかをターゲットに集中プロモーション

## 7. 今後の展望（2ndリリース以降）

1stリリースの成功後、「ポータルサイト」としての機能を順次拡充します。

- 📅 **記念日管理機能**：付き合った日、誕生日など重要日の通知
- 📸 **二人のメモリーアルバム**：デート写真をタイムライン化
- 💰 **共同家計簿**：デート費・生活費の分担管理
- 💬 **カップル専用メッセージ機能**：ウィジェット連携で愛情表現
- 🎁 **ギフト提案AI**：記念日に合わせたプレゼント提案
